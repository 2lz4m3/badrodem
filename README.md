# badrodem

`badrodem` is an anagram of __"bom-adder"__.

It adds an UTF-8 [BOM](https://en.wikipedia.org/wiki/Byte_order_mark) (Byte Order Mark, `0xEF 0xBB 0xBF`) to the beginning of the text file.

## Usage

```
badrodem [file ...]
```
